# NomadForce Website

NomadForce is a modern, responsive landing page website built using **HTML**, **CSS**, and a small amount of **JavaScript**. The design focuses on a creative studio / digital agency style layout with a hero section, portfolio showcase, and responsive navigation for desktop and mobile devices Designed and developed by Eng. Maxamed.Muumin & Eng. Abdukadir Gardhub as part of our coursework at Dugsiiye.

---

# Features

* Responsive navigation bar with **desktop and mobile menu**
* Hamburger menu with toggle animation (open / close icons)
* Full-screen **hero section** with background image and overlay
* Clean grid-based content layout
* Portfolio section with image cards
* Google Fonts and Font Awesome icons integration
* Mobile-first responsive design using media queries

---
# 
Technologies Used

* **HTML5** – Website structure
* **CSS3** – Styling, grid layouts, responsiveness
* **JavaScript (Vanilla)** – Mobile menu toggle
* **Google Fonts** – Inter, Noto Sans, Poppins
* **Font Awesome** – Icons

# Getting Started

To run this project locally:

1. Download or clone the project files
2. Make sure `index.html` and `style.css` are in the same folder
3. Open `index.html` in your browser

No additional setup or build tools are required.

---

# Responsive Design

* **Desktop:** Full navigation menu with logo and sign-up button
* **Tablet & Mobile:**

  * Hamburger menu appears
  * Slide-in mobile navigation
  * Stacked grid layouts for content and portfolio

---

# Customization

You can easily customize the website by:

* Changing images (Unsplash URLs or local images)
* Updating colors in `style.css`
* Editing text content in `index.html`
* Adjusting grid layouts and spacing
